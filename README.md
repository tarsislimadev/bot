# Bot

Core for bots. Many functions.

## run

```bash
docker run -it --rm --net host -e NEWS_API_KEY=<key>   tmvdl/ai:bot
```

## functions

- [ ] news apis

```markdown
/news <search>
```

- [ ] weather apis

```markdown
/weather <city/region>
```

- [ ] delivers apis

```markdown
/ifood <flavor>
```

- [ ] custom apis

```markdown
/g1 <search>
/google <search>
/duckduckgo <search>
```
